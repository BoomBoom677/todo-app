# Mini projet Todo App

## Fonctionnalités
- Ajouter une tâche
- Modifier une tâche
- Supprimer une tâche
- Afficher les tâches triées par priorité

## Base de données
Importer le fichier `todo_app.sql` dans MySQL.

## Lancement
Placer le projet dans le dossier du serveur local puis ouvrir `index.php`.
